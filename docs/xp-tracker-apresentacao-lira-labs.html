Produto web para RPG via bot

XP Tracker

Uma dashboard simples para acompanhar nível, XP restante, progresso diário e rotas de farm em jogos de RPG no Telegram.

O que ele resolve

Jogadores costumam controlar XP por mensagens, prints e contas manuais. O XP Tracker centraliza esse processo em uma experiência visual, rápida e segura.

Clareza quanto falta para upar
Farm runs de cripta e masmorra
Nuvem salvo com login Google

Menos conta manual. Mais direção para evoluir.

O usuário pode testar como visitante e salvar o progresso depois, quando decidir entrar com Google.

Features essenciais

Acesso

Visitante + Google

Teste sem cadastro, painel do plano e migração do progresso para a conta quando fizer login.

Dashboard

Progresso visual

Nível, XP total acumulado, XP restante, percentual, jornada de conquistas e meta diária.

Farm

Criptas e masmorras

Registro rápido de runs com criptas por faixa de nível e masmorras com XP fixo por quantidade de jogadores.

Histórico

Registro inteligente

Salva progresso manual, runs aplicadas, edição e exclusão de registros.

Gráfico

Evolução temporal

Visualização do XP restante e leitura de ritmo recente.

Conquistas

Marcos de uso

Segurança e tecnologia

Base de confiança

Login

Google OAuth

Autenticação via Supabase Auth. Visitante não grava na nuvem até escolher salvar.

Dados

Row Level Security

Cada usuário autenticado só acessa o próprio progresso salvo.

Entrega

Cloudflare Pages

Hospedagem com HTTPS, deploy controlado e headers de segurança.
